Css?


Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language like HTML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

Nice Design


Having a "nice" design nowadays is very easy. You just have to download a template from any of the hundreds of template websites on the market. However, differentiation is vital, more and more, and so we must avoid, as much as we can, standards and repeat formulas. We have to surprise our audience from the first second.

Responsive design


Responsive design: The growing mobile first trend in which the mobile takes precedence over other devices has also affected web design. Using websites with responsive designs that allow content to be viewed from any device improves the user experience and avoids an increase in the rebound rate.

How web design will be in 2019


Trends in web design in 2019 do not invent the wheel as much as they settle or accentuate previous trends. After all, the goal of any website should be to offer the user what they are looking for and guide them in the most friendly way possible.
